# Applied Bioinformatics Capstone Project

## Overview
The project repository is the primary location for my BINF7700 Capstone project. The repository will consist of planning documents, data, and analysis, paperwork, and final materials prepared during the course of the term.

This master's project is centered on the use of bioinformatics and data analysis techniques for the exploration of biological datasets in a reproducible and environment for ethical research.
